MPEG-x are a set of standards and formats.
For example, MPEG-1 is a set of standards like MPEG-1 Part 2 (video), MPEG-1 Part 3 (audio) etc.
Likewise for MPEG-2 and MPEG-4 set of standards.
See Wikipedia: MPEG template page

The test media files were created using FFmpeg 7.1 with no custom settings (like profiles, color spaces, SDR/HDR, etc.)
Example: ./ffmpeg.exe -i base.mp4 -c:v h264 -c:a mp3 h264-mp3.mp4
To create audio-only output add -map 0:a option
To get a list of supported codecs in FFmpeg: ffmpeg -codecs
To get a list of supported formats in FFmpeg: ffmpeg -formats

  1. MP2 is actually an MPEG (either MPEG-1 or MPEG-2) file container with MPEG-1 Audio Layer II (MP2) encoding.
    In other words, its container is just MPEG and the audio codec is MP2
  2. MP3 is actually an MPEG (either MPEG-1 or MPEG-2) file container with MPEG-1 Audio Layer III (MP3) encoding.
    In other words, its container is just MPEG and the audio codec is MP3
  3. MP4 is also called MPEG-4 Part 14 and is derived from QTFF
  4. MPEG-1 Part 2 is also called MPEG-1 Video
  5. H.262 is also called MPEG-2 Part 2 or MPEG-2 Video
  6. H.263 is also called MPEG-4 Part 2
  7. QTFF (QuickTime File Format) is almost identical to MP4
  8. FLAC is both a format and a codec
